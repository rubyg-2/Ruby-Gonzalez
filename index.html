Autonomous Maze-Navigating Robot

This was a semester-long team project focused on designing, building, coding, testing, and demonstrating an autonomous robot. Working with a team of three other members, we developed a robot capable of navigating a maze to transport a disabled person while adhering to traffic control rules.

Key Features of the Robot:

  • Color Sensor: Used to detect path indicators.
  • Red Path Indicator: The robot stops for a second before continuing.
  • Blue Path Indicator: Switches to manual mode after beeping twice, allowing a teammate to assist with passenger transfer.
  • Yellow Path Indicator: Designated drop-off zone, switching to manual mode for final assistance.

The entire system was programmed and controlled using MATLAB, ensuring seamless autonomous and manual transitions.

  • Robotics Design & Prototyping
  • Sensor Integration
  • Control System Programming
  • State-Based Navigation
  • Team Collaboration
